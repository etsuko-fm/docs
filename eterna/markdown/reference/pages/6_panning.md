# Panning
![screenshot of panning page](https://d21ikh70ytbexu.cloudfront.net/docs/eterna/260501/panning-lfo-on.png)

The Panning page controls the stereo position of all six voices. Voices are evenly spread within a configurable width, and can be continuously rotated through the stereo field using Twist.



#### Controls
| Control | Function | Description |
|:--------|:---------|:------------|
| K2 | Toggle LFO on/off | Enables or disables the LFO |
| K3 | Select LFO waveform | Chooses the waveform shape for the LFO |
| E2 | Twist / LFO rate | Controls the panning of all voices at the same time; or when LFO is on, LFO rate |
| E3 | Width | Maximum stereo width |
